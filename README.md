# About

Dockerfile to build a pgplot image based on Ubuntu.

# How to run

Just type the following commands

```
$ make
$ docker run -it shaoguangleo/ubuntu-pgplot[:version]
```

As we all know, the version can be `lateset` or `$ cat VERSION`
